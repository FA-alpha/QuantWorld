# QuantWorld 需求文档

## 1. 项目目标

构建一个基于多智能体技术的加密货币市场仿真系统，用于：
- 预测市场情绪演化
- 模拟各类市场参与者行为
- 生成交易策略参考

---

## 2. 需求确认 ✅

| 项目 | 确认内容 |
|------|---------|
| **目标币种** | BTC, ETH, SOL, DOGE, XRP, ADA, BCH（7个主流币），需支持扩展 |
| **美股支持** | 暂不支持，后续扩展 |
| **仿真时间粒度** | 1 小时 |
| **预测输出** | 情绪指标 + 价格区间预测 + 概率分布（丰富输出） |
| **前端界面** | 需要，参考 MiroFish 设计，展示智能体交互关系 |
| **实时行情** | 交易所 API（OKX/Binance） |
| **新闻数据** | Web Search |
| **链上数据** | Coinglass API |

## 4. 智能体角色设计

### 4.1 核心角色（P0）

#### 散户（Retail Trader）× 3
- **乐观型**：FOMO 心理强，倾向于追涨
- **中性型**：相对理性，参考多方信息
- **恐慌型**：风险厌恶，容易割肉

#### 量化交易员（Quant Trader）× 1
- 纯数据驱动，基于技术指标决策
- 关注 MA、RSI、MACD 等指标

#### 巨鲸（Whale）× 1
- 大资金，能影响市场
- 利用散户情绪反向操作

#### KOL（Opinion Leader）× 1
- 发布市场分析
- 影响散户情绪

#### 新闻源（News Feed）× 1
- 注入外部事件
- 触发市场反应

#### 链上分析师（On-chain Analyst）× 1
- 基于 Coinglass 数据分析
- 关注巨鲸动向、爆仓数据、交易所流动

### 4.2 扩展角色（P1）

- 机构交易员
- 做市商
- 套利交易员

## 5. 数据接口

### 5.1 Coinglass API

需要支持的数据：
- 交易所持仓量（Open Interest）
- 多空比（Long/Short Ratio）
- 爆仓数据（Liquidation）
- 资金费率（Funding Rate）
- 交易所流入流出

### 5.2 交易所 API

- 实时价格
- K 线数据（1h）
- 成交量

### 5.3 Web Search

- 加密货币相关新闻
- 市场情绪关键词

## 6. 仿真流程

```
每小时一个时间步：

1. [数据层] 获取最新数据
   - 实时行情（7 个币种）
   - 链上数据（Coinglass）
   - 新闻搜索（Web Search）

2. [新闻源] 注入重要事件（如有）

3. [链上分析师] 分析链上数据
   - 输出：链上信号（巨鲸动向、爆仓风险等）

4. [KOL] 观察市场 + 链上信号，发布分析
   - 输出：市场观点

5. [散户们] 观察市场 + KOL 意见，做出决策
   - 输出：交易决策

6. [量化] 观察技术指标，做出决策
   - 输出：交易决策

7. [巨鲸] 观察市场 + 散户情绪，做出决策
   - 输出：交易决策

8. [聚合层] 汇总所有决策，生成预测
   - 情绪指标
   - 价格区间
   - 概率分布

9. [前端] 实时展示交互过程和预测结果
```
